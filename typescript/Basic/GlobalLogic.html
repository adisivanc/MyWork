In TypeScript, when you have multiple TypeScript files (`*.ts`) and you want to combine them into a single JavaScript file (`*.js`), you can use relative and non-relative module import paths. Here's a brief explanation of both: ### Relative Module Import: - **Relative Path:** It starts with `./` (dot-slash) or `../` (dot-dot-slash). - **Usage:** Used when importing a module from a file located in the same directory or a subdirectory. Example:

        ```typescript // File: moduleA.ts
        export const valueA = 10; // File: 


        moduleB.ts 
        import { valueA } from './moduleA'; 

        console.log(valueA); ```

### Non-relative Module Import: - **Non-relative Path:** It does not start with `./` or `../`. - **Usage:** Used when importing a module from a package or a module installed via npm/yarn. Example:

        ```typescript // File: moduleC.ts 
        export const valueC = 20; // File:


        moduleD.ts 
        import { valueC } from 'moduleC'; 

        // Assuming 'moduleC' is a package or module installed via npm/yarn console.log(valueC); ```

### Combining Multiple TypeScript Files into a Single JavaScript File: Assuming you have multiple TypeScript files (`moduleA.ts`, `moduleB.ts`, `moduleC.ts`, `moduleD.ts`), you can use the TypeScript compiler (`tsc`) to compile them into a single JavaScript file. 1. **Using TypeScript Compiler (`tsc`):** Create a `tsconfig.json` file with the following content:

        ```json { "compilerOptions": { "outFile": "output.js", "module":
        "commonjs" }, "include": ["**/*.ts"] } ```

Then run: ``` tsc ``` This will compile all TypeScript files and generate a single `output.js` file. 2. **Using `--out` Flag Directly:** Alternatively, you can use the `--out` flag directly without a `tsconfig.json` file:

        ``` tsc --out output.js moduleA.ts moduleB.ts moduleC.ts moduleD.ts ```

This will also compile the TypeScript files into a single `output.js` file. Make sure to adjust the module paths accordingly based on whether they are relative or non-relative.
