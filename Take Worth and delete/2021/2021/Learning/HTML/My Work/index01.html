The some() method checks if any of the elements in an array pass a test (provided as a function).

The some() method executes the function once for each element present in the array: If it finds an array element where the function returns a true value, some() returns true (and does not check the remaining values) Otherwise it returns false

Note: some() does not execute the function for array elements without values.
Note: some() does not change the original array.
