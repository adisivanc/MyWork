The following code for the reference


Presentational icons that are hidden from screen readers are coded as:

<svg aria-hidden="true" role="presentation" class="icon icon-bookmark">
<use xlink:href="../../img/icons.svg#icon-bookmark"></use>
</svg>

If you need to provide accessible text for screen readers, the code is:

<svg role="img" class="icon icon-bookmark">
<title>Bookmark</title>
<use xlink:href="../../img/icons.svg#icon-bookmark"></use>
</svg>

Interactive icons that perfom a function should be coded as:

<a href="#" aria-label="Bookmark" role="button">
<svg aria-hidden="true" class="icon icon-bookmark">
<use xlink:href="../../img/icons.svg#icon-bookmark"></use>
</svg>
</a>

<button aria-label="Bookmark">
<svg aria-hidden="true" class="icon icon-bookmark">
<use xlink:href="../../img/icons.svg#icon-bookmark"></use>
</svg>
</button>
